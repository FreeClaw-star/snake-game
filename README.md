# 🎮 极简游戏中心

这个项目收集了几个简单有趣的网页游戏，旨在提供纯粹的、无须下载的游戏体验。

## 🚀 包含的游戏

### 1. 🐍 贪吃蛇 (Snake)
经典的贪吃蛇游戏，采用 HTML5 Canvas 开发。
- **多种控制模式**：支持 👆 触控、📱 摇盘（重力感应）和 ⌨️ 键盘（WASD/方向键）。
- **难度选择**：提供简单、中等、困难三个级别。
- **特性**：最高分记录、实时加速、动态障碍物。

### 2. ⚔️ 自走棋对战 (Auto Chess)
一个精简而完整的自走棋模拟对战游戏。
- **核心机制**：包含棋子购买、三合一升级、多重羁绊组合。
- **英雄系统**：玩家可以操控国王角色，支持近战和远程模式切换。
- **实时特效**：丰富的战斗动画和伤害数字反馈。

## 🛠 如何开始

本项目所有游戏均基于纯前端技术实现，无需配置复杂的开发环境。

1. 克隆或下载本项目。
2. 直接在浏览器中打开根目录下的 `index.html`。
3. 从游戏门户界面选择你想玩的游戏。

## 📁 项目结构

- `index.html`：项目门户入口。
- `snake/`：贪吃蛇游戏文件夹。
- `auto-chess/`：自走棋游戏文件夹。
